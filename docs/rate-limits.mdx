# Limites de taux

Cette section explique les limites de taux de notre API, qui contrôlent le nombre de requêtes qu'un utilisateur peut effectuer dans un délai donné.

## Limites de taux par utilisateur

Chaque utilisateur peut effectuer une certaine quantité de requêtes par minute, par jour, par semaine et par mois. Ces limites sont configurables dans votre tableau de bord developpeur.

## Limites de taux globales

Toutes les requêtes doivent respecter les limites de taux globales de notre API, qui sont configurables dans votre tableau de bord developpeur.
