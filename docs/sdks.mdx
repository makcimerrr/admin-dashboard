# SDKs

This section details the SDKs available for interacting with our API in various programming languages.

## SDKs disponibles

### `/sdks`

- **Méthode**: `GET`
- **Description**: Récupère une liste de toutes les SDKs disponibles.
- **Paramètres**:
- `limit` (optionnel) : Nombre maximum de résultats par page.
- `offset` (optionnel) : Décalage pour la pagination.
- `filter` (optionnel) : Filtre les résultats en fonction de critères spécifiques.
- **Exemple de réponse**:

<CodeBlock
    language="JSON"
    code={`[
  {
    "id": 1,
    "name": "SDK 1",
    "description": "Description de la SDK 1"
  },
  {
    "id": 2,
    "name": "SDK 2",
    "description": "Description de la SDK 2"
  }
]`}
/>
